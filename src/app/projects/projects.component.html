Movie Scraper

Scrape theater, movie and time from IMDB using python BeautifulSoup4 library.
Clean scraped data and use as REST API Endpoint.
Initial GET request returns JSON array of all nearby movie theatres, the movies and their showtimes.

Technologies Used
  • Python
  • BeautifulSoup4
  • Flask
