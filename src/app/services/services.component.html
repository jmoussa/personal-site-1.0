Services

  • UI/UX Design

    UI/UX design involves planning the layout and structure of the web application, how it will look on mobile and how the user will interact with various components. Once that is decided, I generate a mockup (or blueprint) of the site, usually on paper or Adobe XD.

  • Front End Development

    The front end is what the user will see and interact with. This involves building out the display (ideally so that it matches the mockup). I focus on dynamic, interactive, responsive and intuitive web interfaces for both web and mobile.

  • Back End Development

    The back end is where all the logic that connects to your business or brand will go. Sometimes this is not necessary though, if all you want is to display static information for example.
    You will require a back end if your site integrates with:

    • A Database (or anywhere you store your electronic records)
    • 3rd Party Service (Google Maps, Email Service, etc.)
    • Another CMS

  • Database Services

    The database is what will store all of your information in bulk. It's another optional feature and not always necessary depending on the site's desired functionality.
    My database services include:
    • Building a database from Scratch
    • Database Migration
    • Connecting and integrating a database with the back end
