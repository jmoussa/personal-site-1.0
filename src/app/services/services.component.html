Services

  • UI/UX Design

    UI/UX design involves planning the layout and structure of the web application, how it will look on mobile and how the user will interact with its various components.

  • Front End Development

    The front end is what the user will see and interact with. This involves building out the display (ideally so that it matches the mockup). I focus on dynamic, interactive, responsive and intuitive web interfaces for both web and mobile.

  • Back End Development

    The back end is where all the logic that connects to your business/brand will go. Sometimes this is not necessary. For example, if all you want is to display static information that will not change frequently or dynamically.
    You will require a back end if your site integrates with:

    • A Database (or anywhere you store your electronic records)
    • 3rd Party Service (Google Maps, Email Service, etc.)
    • Another CMS (Content Management System)

  • Database Services

    The database is what will store all of your information in bulk. This is another optional feature and not always necessary depending on the site's desired functionality.
